# libmemleak
LibMemleak- A C library for checking Memory Leak.

# Build from source
1. Get source code
```
  $ git clone --recursive git@github.com:Xphobia/libyamlc.git
  $ cd libyamlc
```
2. Install libbacktrace
 ```
  $ cd libbacktrace
  $ ./configure --enable-shared
  $ make
  $ make install
 ```
 3. Compile libyamlc
 ```
  $ make
 ```
